# Stripe Task Project

This is a **frontend project** for integrating **Stripe Checkout** to handle payments securely. It provides a smooth user experience for selecting products, adding them to a cart, and completing payments via Stripe.

## Project Overview
This project demonstrates a **Stripe Checkout integration** in a React frontend application. Users can:
- View a list of products
- Add products to the cart
- Checkout securely via Stripe
- Receive success/failure messages for payment

---

## Features
- Responsive **product list** and **cart UI**
- **Stripe Checkout integration**
- **Payment status handling**
- Supports multiple products and quantities
- Clear, user-friendly interface

---

## Technologies Used
- **Frontend:** React.js, CSS, HTML
- **Payment Gateway:** Stripe
- **State Management:** React Context API

## Installation

1. Clone the repository:

```bash
git clone https://github.com/simmi-verma/Stripe-task.git
cd Stripe-task/frontend


2. Install dependencies:

npm install


3. Start the development server:

npm start

The app should open on http://localhost:3000.
